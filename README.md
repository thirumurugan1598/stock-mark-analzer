🚀 EquitiQ — AI Investment Intelligence Platform

EquitiQ is an AI-powered stock market analysis web application that provides real-time insights, sentiment analysis, and intelligent BUY/HOLD/SELL signals for NSE & BSE stocks.

It combines technical indicators, fundamental data, and news sentiment into one powerful dashboard to simplify investment decisions.


✨ Features
📊 Live Market Dashboard (NIFTY, SENSEX, BANK NIFTY, USD/INR)
🤖 AI Stock Analyzer
🧠 Self-learning AI Training System
📰 Real-time News Sentiment Analysis
📈 Technical Indicators (RSI, MACD, SMA, Bollinger Bands)
🔐 Secure (AES-256, No Data Storage)
